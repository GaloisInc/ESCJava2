Syntax Summary

P ::=                   processes
  new n_1 ... n_k. P      new names in a restricted scope P
  -                       inactive process (often omitted)
  P | P'                  parallel composition
  rec x. P                recursive process
  t[P]                    ambient named t containing P
  do t. P                 a capability-based action
  say t                   output t into current ambient
  ask x. P                input x from current ambient, then P
  #                       kills the current ambient
  "...". P                writes "..." on console, then P
  wait r. P               wait r (real) seconds, then P
  be t. P                 rename the current ambient t, then P
  let x(x_1 ... x_k) = P; P'  define a process P named x in the
                              scope P', parameterized on the x_i
  x(t_1 ... t_k)          the expansion of a definition
  (P)                     grouping

t ::=                   terms
  x                       identifier (new-bound or ask-bound)
  "..."                   character string
  r                       real number
  in x                    in capability
  out x                   out capability
  open x                  open capability
  stay                    empty route
  t & t                   composite route
  (t)                     grouping
